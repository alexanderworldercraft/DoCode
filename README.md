# DOCODE

Documentation Organisee pour les Commandes et le CODE.

DOCODE est une application HTTP destinee a centraliser, structurer et consulter de la documentation technique liee aux commandes, au code, aux projets et aux procedures de developpement.

## Objectif

L'objectif initial est de poser une base claire pour une interface de documentation composee de deux parties :

- un backend HTTP pour exposer les donnees et les futures API ;
- un frontend React pour naviguer dans la documentation.

## Stack cible

### Backend

- Node.js
- Fastify
- Prisma
- MySQL

### Frontend

- React
- React Router
- Tailwind CSS
